# Changelog

All notable changes to `zego-audio-video-calling` will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Complete audio and video calling functionality
- Web application support with beautiful UI
- Mobile API endpoints for iOS and Android
- Push notification support (FCM and APNs)
- Real-time WebSocket broadcasting
- Call history and management
- Online/offline status tracking
- Laravel 10-12+ compatibility
- Auto-discovery service provider
- Comprehensive documentation
- Mobile integration guide
- Customizable routes and middleware
- Configuration options for all features

### Features
- Zero-configuration setup (just add credentials to .env)
- Cross-platform calling (Web ↔ Mobile, Mobile ↔ Mobile)
- Beautiful, responsive call interface
- Call state synchronization across devices
- Device token management
- User availability checking
- Call duration tracking
- Configurable call settings
- Security features

### Documentation
- Complete README with examples
- Mobile integration guide for iOS and Android
- API documentation
- Configuration guide
- Troubleshooting section
